### Setup Experiment Environment


#### 1 Sanity Check on our patches:
```
./setup.py defects4j
```

#### Test patches
```
./autotest.py <patch name>  <correct|plausible> <ASE15|EMSE18> <evosuite|randoop>
i.e. ./autotest.py patch1-Chart-1-CapGen.patch correct ASE15 randoop
```
